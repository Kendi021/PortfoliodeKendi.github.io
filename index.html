Réseau & Infrastructure

Serveur VPN automatisé

Déploiement complet d'un serveur OpenVPN sur Ubuntu, avec provisioning automatique via cloud-init et installation non-interactive basée sur le script angristan/openvpn-install.

  • Cloud-config Ubuntu : locale, stockage LVM, authentification SSH par clé Ed25519, accès root désactivé.
  • Pare-feu UFW préconfigué (SSH + UDP 1194) et Fail2ban activé dès le premier démarrage.
  • Script Bash idempotent : protocole UDP, port 1194, génération automatique du profil client client-admin.ovpn.
  • Infrastructure PKI complète (CA, certificat serveur, clé Diffie-Hellman, TLS 1.3 / AES-256-GCM).

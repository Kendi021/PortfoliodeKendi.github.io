Projet OpenVPN sécurisé

Projet pédagogique centré sur la mise en place d'un accès distant sécurisé. L'objectif était de déployer un serveur VPN, standardiser l'installation et produire une configuration réutilisable.

  • Script shell d'installation automatique basé sur le projet angristan/openvpn-install.
  • Paramètres définis pour un déploiement cohérent: protocole UDP, port 1194, client administrateur et configuration non interactive.
  • Production d'une fiche projet PDF et d'un script prêt à rejouer pour faciliter la démonstration.
